# Ashfault Documentation

The purpose of this document is to serve as a developer's guide for any software using the Ashfault framework.
All relevant functions and module are documented here.

## Building

### Linux

To build the Ashfault project, simply run the script located in `scripts/build.sh`.

This will produce a `build` folder with a Makefile. You can then simply run `make` inside this folder to build the project.
